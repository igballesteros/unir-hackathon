# ⚡ START NOW - 2 Minutes to Access

## Step 1: Start Backend (Terminal 1)
```bash
cd backend
npm run dev
```
✅ Wait until you see "Server running on port 3000"

## Step 2: Start Frontend (Terminal 2 - NEW terminal)
```bash
cd frontend
npm run dev
```
✅ Wait until Vite shows:
- **Local:** http://localhost:5173
- **Network:** http://192.168.x.x:5173  ← **SHARE THIS URL!**

## Step 3: Share the Network URL
Copy the "Network" URL from Terminal 2 and share it!

**Anyone on the same WiFi can access it!**

---

## If you want PUBLIC internet access (requires ngrok):

1. Install ngrok: `brew install ngrok` (takes 1-2 min)
2. Then run: `./start-ngrok.sh`
3. Share the ngrok URL

**But the Network URL works NOW for same WiFi!**
